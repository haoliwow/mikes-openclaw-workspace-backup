## 一通數位 #52 Java Developer - 詳細送人條件

### 必須符合
- ✅ Java Backend 開發經驗 **3–5 年**
- ✅ 熟悉 **Spring Boot / Spring Framework**
- ✅ 有 **REST API 開發經驗**
- ✅ 有 **資料庫使用經驗（MySQL / SQL）**

### 符合身份
- Backend Engineer
- Java Developer
- Software Engineer

### 不符合身份（DISQUALIFY）
- ❌ QA Engineer
- ❌ System Analyst / PM
- ❌ DevOps Engineer (unless Java dev background)
- ❌ 培訓班出身無實務經驗
