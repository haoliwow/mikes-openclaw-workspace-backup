# 開發信範本 — 集團合併年報表 + 向上報告線版本

**適用場景：** 有海外子公司、集團合併報表、上市準備經驗的財務人選

**風格特徵：**
- 主題行：「業務經驗關鍵字 + 集團財務經驗｜新機會 + 高級別報告線」
- 開場：誇獎具體的技術背景（不是籠統讚美）
- 中段：說明新職位的「特色與層級」（向 Group CFO 報告等）
- 結尾：誠懇交流邀請（不是高壓銷售）

---

## 範本

**主題行：**
```
閱讀財務 + 集團合併年報表經驗｜想與妳交流一個小而高薪財務主管機會
```

**內文：**
```
Hi Pei-Chun 您好，

在看妳的履歷時，有幾個地方讓我印象深刻。

特別是妳在 Handa Pharmaceuticals 不僅做完整的海外子公司帳務管理（美國、杭州），也負責月度集團合併財報與美國稅務申報。但最亮眼的是——妳有上市準備經驗，親身參與過招股說明書、財務預測這種層級的工作。這在台灣財務人中並不常見，足見妳的能力被高度信任。

剛好我最近正在協助一家瑞典 Nasdaq 上市公司，在台灣成立子公司。他們在找一個 Finance Supervisor，直接向 Group CFO 報告。主要負責集團財務報告、跨國稅務規劃、全球現金流協調，以及往後台灣子公司的財務系統建立與控制。

這個職位有幾個特色：
- **責任獨立**：不是坐在大公司下面跑流程，而是妳主導台灣財務系統的架構
- **層級清楚**：向 Group CFO 直接報告，代表這是集團層級的角色
- **國際協作**：與瑞典母公司跨時差協調，也會接觸新加坡、其他地區的業務
- **未來彈性**：如果台灣業務成長或上市融資，妳會是核心決策者

我知道妳在 Handa 穩定發展多年，但想單純交流一下——如果有機會主導一個從零建立的財務系統、在上市公司的國際環境中工作，妳會有興趣嗎？

有時間聊聊嗎？

—— Phoebe
Step1ne
```

## 使用場景

**適合對象：**
- 有 5 年+ 財務經驗
- 海外子公司或集團背景
- 上市公司或上市準備經驗
- 現任財務 Manager / 高級別
- 追求「責任獨立」而非單純「職位升級」

**調整點（保持框架，替換細節）：**
1. **主題行**：換成「該人的核心經驗 + 新職位層級」
2. **開場段**：替換為該人的具體成就（合併財報、稅務申報、上市經驗等）
3. **中段—職位描述**：根據實際公司與職位調整「報告線、責任、國際環節」
4. **特色段**：保持 4 點邏輯，根據職位調整內容
5. **結尾**：保持溫暖交流語氣，強調「與妳交流」而非推銷

---

## 核心語氣要點

✅ **做什麼：**
- 誇獎具體的技術能力（合併財報、稅務申報、系統導入）
- 說明職位的「層級與獨立性」（向上報告線、責任邊界）
- 強調「國際環境」和「系統建造」的成長機會
- 誠懇邀請「交流」而非「面試」

❌ **不做什麼：**
- 不說「妳很優秀」（要說具體的技術成就）
- 不高壓推銷職位（要邀請對話）
- 不只談薪資或頭銜（要談成長空間與責任獨立）
- 不過度強調「急招」（要營造稀有機會感）
